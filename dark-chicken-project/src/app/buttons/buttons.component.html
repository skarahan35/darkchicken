Darkchicken Buttons



DarkChicken is an innovative library providing user interface tools for developers using Angular. The "Button" UI elements in this library are designed to optimize user experience and facilitate the developers' work. Here are the basic features and usage of DarkChicken Button elements:
Basic Features
Various Styles: DarkChicken Button elements come in various styles. Predefined styles such as default, primary, secondary, and danger allow you to use buttons for different purposes in your projects easily.
Customizability: You can customize the appearance and feel of the Button elements according to your projects. You can change the color, size, border radius, and more via CSS.
State Management: Button elements support different states such as active, disabled, and loading. These states allow you to manage user interactions and feedback more effectively.
Accessibility: DarkChicken is designed according to accessibility standards. Button elements are fully compatible with screen readers and keyboard navigation.

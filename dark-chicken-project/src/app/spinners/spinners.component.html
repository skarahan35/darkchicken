Darkchicken Spinner



DarkChicken is an innovative library providing user interface tools for developers using Angular. The "Spinner" UI elements in this library are designed to optimize user experience and facilitate the developers' work. Here are the basic features and usage of DarkChicken Spinner elements:
Basic Features
Various Styles: DarkChicken Spinner elements come in various styles. Predefined styles such as primary, secondary, success, danger, warning, info, light, and dark allow you to use spinners for different purposes in your projects easily.
Customizability: You can customize the appearance and feel of the Spinner elements according to your projects. You can change the color, size, speed, and more via CSS.
State Management: Spinner elements support different loading states. These states allow you to manage user interactions and feedback more effectively.
Accessibility: DarkChicken is designed according to accessibility standards. Spinner elements are fully compatible with screen readers and keyboard navigation.
